# 📚 Udemy Course Clone (HTML & CSS)

A static front-end project that replicates the design of Udemy’s course pages using **only HTML and CSS**.  
This project focuses on layout, styling, and responsiveness without any JavaScript functionality.

---

## 🚀 Features
- Responsive homepage inspired by Udemy
- Navigation bar with logo, search bar, and links
- Hero section with featured courses
- Grid-based layout for course cards
- Course detail page design (static)
- Footer with links and social icons

---

## 🛠️ Tech Stack
- **HTML5** → Structure and semantic markup  
- **CSS3** → Styling, Flexbox, and Grid for layout  
- **Responsive Design** → Media queries for mobile, tablet, and desktop
